# AGENTS.md

## Cursor Cloud specific instructions

This repository (`Induction-Y25`) is a **documentation-only student assignment repository**. It contains markdown writeups, screenshots, PDFs, and text files submitted by students for three induction tasks (GitHub basics, OverTheWire Bandit wargame, and a SmartBank C++ design spec).

### Key facts

- **No source code exists** — there are zero `.cpp`, `.py`, `.js`, or other source files.
- **No build system** — no `Makefile`, `CMakeLists.txt`, `package.json`, or similar.
- **No tests or CI/CD** — no test framework, no `.github/workflows`, no CI pipeline.
- **No runnable application** — Task 3 (SmartBank Enterprise System in C++) is described in `README.md` as a specification for students to implement, but no implementation has been committed.
- **No dependencies to install** — the update script is intentionally a no-op (`true`).

### Development workflow

Since this is a documentation repo, the typical workflow is:
1. Fork the repo and create a student-named branch.
2. Add assignment submission files (markdown, images, PDFs).
3. Open a PR back to the main repo.

There is nothing to lint, test, build, or run.
